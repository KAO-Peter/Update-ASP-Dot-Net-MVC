## 🛠️ 技術架構

### 核心元件
- **HRPortal.Core**: 業務邏輯核心
- **HRPortal.Core.Contracts**: 介面定義
- **HRPortal.DBEntities**: 資料實體
- **HRPortal.Services**: 業務服務層

### 技術棧
- **.NET 8.0**: 現代化執行環境
- **Entity Framework Core**: 資料存取層
- **ASP.NET Core MVC**: Web 框架
- **Repository Pattern**: 資料存取抽象
- **Dependency Injection**: 依賴注入容器
